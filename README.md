# 대규모 유럽 축구 데이터 기반 리그·팀·선수 성과 패턴 분석

## 1. 프로젝트 개요

본 프로젝트는 공개 축구 데이터셋인 European Soccer Database를 수집하여 HDFS에 저장하고, Spark와 Hive를 활용해 유럽 축구 리그별 경기 패턴, 팀 성과 요인, 선수 능력치 패턴을 분석하는 빅데이터 처리 시스템을 구현하는 것을 목표로 한다.

축구 경기의 결과는 단순히 득점 수만으로 설명되기 어렵다. 리그별 경기 스타일, 홈/원정 여부, 팀의 공격·수비 특성, 선수 능력치와 같은 다양한 요인이 복합적으로 작용한다. 따라서 본 프로젝트에서는 여러 시즌과 여러 리그에 걸친 축구 데이터를 통합적으로 분석하여 경기 결과와 성과에 영향을 주는 주요 패턴을 확인하고자 한다.

---

## 2. 문제 정의

본 프로젝트에서 풀고자 하는 문제는 다음과 같다.

### 2.1 풀고자 하는 문제

유럽 축구 리그의 경기 데이터, 팀 데이터, 선수 데이터를 활용하여 다음 질문에 답하고자 한다.

1. 리그별 홈 승률, 평균 득점, 무승부 비율은 어떻게 다른가?
2. 팀별 공격력과 수비력 지표는 시즌 성과와 어떤 관계가 있는가?
3. 팀 전술 속성과 득점, 실점, 승률 사이에는 어떤 관련이 있는가?
4. 선수의 연령, 능력치, 평균 평점 사이에는 어떤 패턴이 있는가?

이를 통해 단순한 경기 결과 조회가 아니라, 리그·팀·선수 단위의 데이터를 조인하고 집계하여 축구 경기 성과에 영향을 주는 요인을 분석한다.

### 2.2 사용할 데이터

본 프로젝트에서는 Kaggle의 European Soccer Database를 사용할 예정이다.

- 데이터셋: European Soccer Database
- 데이터 출처: Kaggle
- 원본 형식: SQLite database
- 변환 후 사용 형식: CSV
- 주요 내용:
  - 유럽 축구 경기 정보
  - 국가 및 리그 정보
  - 팀 정보
  - 팀 속성 정보
  - 선수 정보
  - 선수 능력치 정보

주요 사용 테이블은 다음과 같다.

| 테이블 | 설명 |
|---|---|
| Country | 국가 정보 |
| League | 리그 정보 |
| Match | 경기 정보 |
| Team | 팀 정보 |
| Team_Attributes | 팀별 전술 및 속성 정보 |
| Player | 선수 기본 정보 |
| Player_Attributes | 선수 능력치 및 평점 정보 |

원본 데이터는 용량이 크기 때문에 GitHub에는 직접 업로드하지 않고, 데이터 출처와 수집 방법만 기록한다. 최종 제출 시에는 샘플 데이터만 GitHub에 포함할 예정이다.

## 3. 기술 스택

본 프로젝트에서는 강의에서 다룬 Hadoop 기반 빅데이터 처리 기술을 중심으로 다음 도구를 사용할 계획이다.

| 구분 | 사용할 도구 | 역할 |
|---|---|---|
| 데이터 수집 | Bash, Python, Kaggle API | 공개 데이터 다운로드 및 수집 자동화 |
| 데이터 변환 | Python, SQLite | SQLite DB를 테이블별 CSV로 변환 |
| 데이터 저장 | HDFS | 원본 데이터, 전처리 데이터, 분석 결과 저장 |
| 데이터 전처리 | Apache Spark, Spark DataFrame | 결측치 처리, 타입 변환, 파생 변수 생성 |
| 데이터 분석 | Spark SQL, HiveQL | 리그별·팀별·선수별 집계 분석 |
| 테이블 관리 | Apache Hive | 전처리 및 분석 결과 테이블 관리 |
| 시각화 | Python, Matplotlib | 분석 결과 그래프 생성 |
| 보조 도구 | Pandas | 최종 결과 CSV 확인 및 시각화 보조 |

핵심 전처리와 분석은 Spark와 Hive를 중심으로 수행한다. Pandas는 핵심 분석 도구가 아니라, Spark/Hive에서 생성된 최종 결과 CSV를 확인하거나 시각화하는 보조 도구로만 사용한다.

## 4. 구현 계획

본 프로젝트는 데이터 수집부터 분석 결과 생성까지 다음과 같은 파이프라인으로 구현할 계획이다.

```text
Kaggle 공개 축구 데이터 다운로드
        ↓
SQLite DB 압축 해제
        ↓
Python을 이용해 SQLite 테이블을 CSV로 변환
        ↓
HDFS raw 디렉토리에 원본 CSV 적재
        ↓
Spark DataFrame 기반 전처리
        - 결측치 처리
        - 날짜/시즌 컬럼 정리
        - 리그/팀/선수 테이블 조인
        - 홈/원정 결과 컬럼 생성
        - 승점, 득점 차이, 평균 득점 등 파생 변수 생성
        ↓
전처리 결과를 HDFS processed 디렉토리에 저장
        ↓
Hive 외부 테이블 생성
        ↓
Spark SQL / HiveQL 기반 분석
        - 리그별 경기 패턴 분석
        - 팀별 성과 지표 분석
        - 팀 속성과 경기 결과 관계 분석
        - 선수 능력치 패턴 분석
        ↓
분석 결과를 HDFS result 디렉토리에 저장
        ↓
Python / Matplotlib을 이용한 시각화
        ↓
최종 보고서 및 발표 자료 작성
